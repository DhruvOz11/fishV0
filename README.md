This application is made for a local seafood wholeseller based in Surat, Gujarat, In. he want's to expand his business horizons B2B to D2C so this is a Production Grade Application, system design built with scalability for future expansion to different cities. this application can also be rebranded and sold to different clients:

the fish merchant is a full-stack quick-commerce web application designed for online seafood ordering with real-time delivery workflows, admin management, and WhatsApp-based order confirmation. It simulates a production-grade system similar to platforms like Licious but tailored for rapid delivery and local operations.
